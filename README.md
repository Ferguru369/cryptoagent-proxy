# CryptoAgent Proxy

A lightweight proxy server that routes CryptoAgent's Binance API calls through a non-restricted server, bypassing Binance's regional IP restrictions on Internet Computer nodes.

## Deploy to Railway

1. Fork or clone this repo to your GitHub account
2. Go to [railway.app](https://railway.app) and log in
3. Click **New Project** → **Deploy from GitHub repo**
4. Select this repo
5. Railway deploys automatically — takes about 60 seconds
6. Go to your project → **Settings** → **Networking** → **Generate Domain**
7. Copy your Railway URL (e.g. `https://cryptoagent-proxy-production.up.railway.app`)
8. Paste it in your CryptoAgent chat and say "update my proxy URL to [your URL]"

That's it — you're live!
